Google Cloud Batchを利用するサンプル
====================

## 手順

1. Batch ジョブを実行するプロジェクトのAPIを有効にする
    - Batch API
    - Artifact Registry API
    - Cloud Build API
1. Artifact Registry リポジトリを作成する
1. Batchジョブスクリプトを作成する

## 使い方

### venv

- activate
    ```fish
    # fishの場合
    source .venv/bin/activate.fish
    ```

- deactive
    ```bash
    deactivate
    ```
